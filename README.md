# ReliabilityEngine-Observability-Suite
A production-ready observability platform for Kubernetes (EKS), implementing the "Three Pillars of Observability" (Metrics, Logs, Traces) This system utilizes OpenTelemetry for instrumentation, centralized storage via Prometheus/Loki/Jaeger, and automated SLO-based alerting. Built using Infrastructure as Code (Terraform) and GitOps (GitHub Actions)
